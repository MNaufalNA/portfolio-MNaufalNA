## Overview

Mengimplementasikan autentikasi dua faktor (2FA) menggunakan kode One Time Password (OTP) yang dikirimkan secara real time melalui WhatsApp API (Fonnte) pada Sistem Informasi Akademik SMAN 3 Purwokerto

## Key Features

- **Two-Factor Authentication (2FA)**: Penerapan mekanisme login dua langkah, di mana pengguna harus memvalidasi kode OTP setelah berhasil memasukkan username dan password untuk mengakses sistem.
- **WhatsApp API Integration**: Integrasi API pihak ketiga (Fonnte) untuk mengirimkan kode OTP secara otomatis ke nomor WhatsApp pengguna.
- **Real-Time OTP Delivery**: Pengujian menunjukkan pengiriman kode OTP berlangsung sangat cepat dengan rata-rata delay hanya 1,2 detik.
- **Secure Code Generation**: Kode OTP 6 digit yang dihasilkan bersifat unik, acak, dan memiliki masa kedaluwarsa untuk meningkatkan ketahanan terhadap serangan brute force.

---

## Technologies Used

- **CodeIgniter (PHP Framework)**: Pondasi utama yang saya pakai untuk membangun seluruh Sistem Informasi Akademik dengan terstruktur.
- **WhatsApp API (Fonnte)**: Layanan pihak ketiga yang menjadi jembatan untuk mengirimkan kode OTP secara otomatis ke WhatsApp pengguna.
- **PHP & MySQL**: PHP saya gunakan untuk mengelola logika di belakang layar (backend) dan proses pengamanan data, sementara MySQL saya pakai untuk menyimpan dan mengelola seluruh data akademik.
- **HTML, CSS, & Bootstrap**: Saya gunakan kombinasi ketiganya untuk merancang antarmuka pengguna (front-end) yang rapi, responsif, dan mudah digunakan.

## Outcome

Implementasi autentikasi dua faktor (2FA) menggunakan kode One Time Password (OTP) yang dikirimkan melalui WhatsApp API (Fonnte) pada Sistem Informasi Akademik SMAN 3 Purwokerto telah berhasil diterapkan secara efektif, terbukti mampu meningkatkan aspek keamanan login sistem. Kinerja sistem ini optimal, ditunjukkan dengan rata-rata waktu pengiriman OTP hanya 1,2 detik, sementara hasil pengujian kepuasan pengguna (berdasarkan Skala Likert) mencapai 82,6%, yang mengindikasikan bahwa pengguna sangat mendukung dan puas dengan fungsionalitas dan kemudahan penggunaan solusi keamanan baru tersebut.
